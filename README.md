# Elowen Blog

这是一个基于 Astro 搭建的个人博客项目。

## 内容

- 博客文章
- 项目记录
- 个人简历
- 技术学习笔记

## 开发

安装依赖：

```bash
pnpm install
```

启动本地开发服务：

```bash
pnpm run dev
```

构建生产版本：

```bash
pnpm run build
```

本地预览构建结果：

```bash
pnpm run preview
```

## 目录

- `src/content/blog`：博客文章
- `src/content/projects`：项目内容
- `src/content/pages`：普通页面
- `src/data/site-config.ts`：站点配置
- `src/assets/images`：图片资源
