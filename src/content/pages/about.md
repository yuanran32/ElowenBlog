## 你好，我是 Elowen

一名前端工程师，目前专注于 Web 应用开发和前端工程化实践。

我相信好的技术文章应该是"写给半年前的自己看的"——把当时困惑的问题、摸索的过程和最终的理解记录下来，既是对自己的复盘，也可能帮到正在经历同样阶段的人。

## 技术方向

- **前端开发**：React / Vue 生态，TypeScript，组件设计与状态管理
- **工程化**：构建工具链、自动化测试、CI/CD、代码质量体系
- **全栈探索**：Node.js 服务端、数据库设计、API 设计
- **开发体验**：工具链优化、开发效率提升、团队协作规范

## 关于这个博客

这个博客基于 Astro 5 + Tailwind CSS v4 搭建，内容使用 Markdown 管理，部署在 GitHub Pages 上。

这里会记录：

- 技术学习过程中的思考和总结
- 项目实践中的架构决策和踩坑经验
- 前端工程化相关的工具和方法论
- 偶尔的阶段性复盘和职业思考

## 写作原则

- 注重实践，避免纯理论堆砌
- 记录决策过程，不只是最终结论
- 代码示例力求可运行、可复现
- 保持诚实，踩过的坑不回避
